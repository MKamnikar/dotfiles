# dotfiles

These are the dotfiles for various programs I use.

---

The dotfiles are for following programs:
* vim/nvim (editor)
* zsh (shell)
* i3wm/i3-gaps (window manager)
* i3-blocks (status bar)
* sxhkd (key binder)
* ranger (file manager)
* dunst (notification manager)
* newsboat (RSS reader)
* compton (compositor)
* zathura (pdf reader)
* autorandr (screen layout manager)
